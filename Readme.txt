- Installer le terminal pour Windows dans le Microsoft Store.
- Installer NodeJS

Lancer automatiquement le Socket au démarrage de Windows :

Dans un terminal :
- npm install ws
- npm install -g qckwinsvc
- qckwinsvc
- remplir le formulaire

Pour désinstaller le service :

Dans un terminal :
- qckwinsvc --uninstall
- remplir le formulaire
